# Account Management

Manage your PodPDF account.

## Endpoints

- `GET /accounts/me` - Get account info
- `GET /accounts/me/billing` - Get current month billing summary
- `GET /accounts/me/bills` - List all bills/invoices
- `GET /accounts/me/stats` - Get PDF count and billing stats
- `PUT /accounts/me/upgrade` - Upgrade plan
- `DELETE /accounts/me` - Delete account
